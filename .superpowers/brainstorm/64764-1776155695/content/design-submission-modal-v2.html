How it works

  • Single view: Form fields at top, "or" divider, drag-and-drop at bottom
  • Compose path: Fill in model + prompt → Submit builds a 1-line JSONL, uploads it, creates batch with completion_window=1h
  • Upload path: Drop a .jsonl → form fields disable/grey out, Submit uploads file and creates batch with completion_window=1h
  • Mutual exclusion: Dropping a file disables the form; clearing the file re-enables it
  • On submit: Modal closes, user returns to Async list
