Changes from v1

  • No toggle — always shows all batches regardless of completion window
  • Type column — always visible. Derived from completion_window: 1h = async, everything else = batch
  • Window column — shows the raw completion_window value (24h, 1h, 4h, etc.) so platform managers see what they set
  • Columns: Created, User, Type, Window, Status, Progress, Duration, Actions
  • Note: 4th row shows a custom 4h window — this is why we can't hardcode
