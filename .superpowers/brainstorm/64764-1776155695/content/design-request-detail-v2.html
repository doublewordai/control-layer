Layout

  • Left column: Input/output — the conversation content, takes most of the width
  • Right sidebar: Metadata in stacked key-value pairs, grouped with dividers
  • Details group: Status, model, created, duration
  • Tokens group: Prompt tokens, completion tokens, cost
  • Batch group: Batch ID (clickable, links to batch detail page), completion window, request index
  • Batch ID link: Navigates to /workloads/batch/:batchId for users who want the full batch view
