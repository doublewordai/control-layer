Details

  • Two modes via tabs: "Upload JSONL" and "Compose"
  • Upload mode: Same file upload flow as current batch creation, but hardcodes completion_window=1h
  • Compose mode: Model selector, system prompt (optional), user prompt, temperature, max tokens
  • Behind the scenes: Compose mode builds a JSONL with one line, uploads it as a file, then creates a batch via POST /ai/v1/batches with completion_window=1h
  • On submit: Modal closes, user returns to Async list where their new request(s) appear
  • No completion window selector: Always 1h for async — this is the key UX simplification
