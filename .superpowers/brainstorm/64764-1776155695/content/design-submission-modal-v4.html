How it works

  • Shared config: Model, system prompt, temperature, max tokens apply to all prompts
  • "+ Add another prompt": Appends a new prompt field (Prompt 2, 3, ...)
  • Remove: Each prompt after the first has a "✕ Remove" link
  • Footer counter: Shows "3 requests" so the user knows how many will be submitted
  • On Create: Builds a JSONL with N lines (one per prompt), uploads as file, creates batch with completion_window=1h
  • JSONL upload: Still available below the "or" divider as an alternative
